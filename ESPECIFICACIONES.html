🌐 Despliegue GitHub Pages

Instrucciones Paso a Paso:

  1. Crear repositorio en GitHub (público)
  2. Clonar o subir archivos locales
  3. Ejecutar comandos git: 
    git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/tu-usuario/pnk-inmobiliaria.git
git push -u origin main
  4. Ir a Settings → Pages
  5. Seleccionar Deploy from branch → main → /(root)
  6. Guardar y esperar despliegue
  7. Sitio disponible en: https://tu-usuario.github.io/pnk-inmobiliaria/
