================================================================================
                        INSTRUCCIONES RÁPIDAS
================================================================================

INSTALAR GIT (si no lo tienes):
  1. Ve a: git-scm.com
  2. Descarga e instala
  3. Reinicia la terminal

CONFIGURAR GIT (primera vez):
  git config --global user.name "Tu Nombre"
  git config --global user.email "tu-email@gmail.com"

SUBIR A GITHUB (desde la carpeta proyecto):
  git init
  git add .
  git commit -m "Inicial: PNK Inmobiliaria"
  git branch -M main
  git remote add origin https://github.com/TU-USUARIO/pnk-inmobiliaria.git
  git push -u origin main

ACTIVAR GITHUB PAGES:
  1. Ve a Settings del repositorio
  2. Busca "Pages" en el menú izquierdo
  3. Selecciona: Deploy from branch → main → / (root)
  4. Save

TU SITIO EN VIVO:
  https://tu-usuario.github.io/pnk-inmobiliaria/
  (Espera 5 minutos después de hacer push)

HACER CAMBIOS:
  git add .
  git commit -m "Descripción del cambio"
  git push
